1) перегрузка:
     при попытке перегрузки метода с одной сигнатурой, но с разными модификаторами либо возвращаемым значением
     выдает ошибку duplicate method "method_name" in Type "class_name" и рекоммендует переименовать метод.
2) переопределение:
  1)
     a)при переопределении метода из суперкласса с понижением модификатора доступа(public->private) пишет:
	    Cannot reduce the visibility of the inherited method from "суперкласс"
     b)при переопределении метода с модификатором private компилятор не видит метод суперкласса и считает одноименный метод
        потомка новым
     c)при переопределении метода из суперкласса с повышением модификатора доступа(protected->public) ошибок нет
     d)при переопределении метода из суперкласса с модификатором доступа по умолчанию будет применятся одно из свойств выше
  	    в зависимости от месторасположения потомка и суперкласса(в одном пакете или нет)
  2)при переопределении метода из суперкласса с заменой возвращаемого типа пишет:
     The return type is incompatible with "суперкласс.имя_метода()"
