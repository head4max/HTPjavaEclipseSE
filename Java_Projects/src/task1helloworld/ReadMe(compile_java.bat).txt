/**
   В общем виде этот bat-фаил принимает 2 параметра:
   имя класса "name" фаила name.java, который изначально должен лежать в корне папки проекта либо в папке ./java
   имя "jarname" фаила jarname.jar, который появится после архивирования байт кода из name.class(который будет скомпилирован командой javac) и манифеста созданного внутри этого батника

   В процессе компиляции и архивирования будут созданы папки "classes","java","lib", в которые соответственно будут перемещены фаилы *.class,*.java,*.jar
   также в корне папки проекта будет сгенерирован manifest.mf
